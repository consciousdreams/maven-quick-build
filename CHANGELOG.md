# Changelog

## [Unreleased]

## [1.0.1]

### Added

- GitHub Actions CI/CD workflows for automated build, verification, and release
- Default keyboard shortcuts for plugin actions (`Cmd+Option+S` / `Cmd+Option+M` on Mac, `Ctrl+Alt+S` / `Ctrl+Alt+M` on Win/Linux)
- JetBrains Marketplace badge and installation link in README
- Vendor metadata in `plugin.xml`

### Changed

- Plugin description moved to dedicated `pluginDescription.html` file

## [1.0.0]

### Added

- Toolbar button to run `mvn clean install -Dmaven.test.skip=true`
- Toolbar button to run `mvn clean install`
