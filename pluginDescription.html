Maven Quick Build

One-click Maven builds directly from the toolbar

Maven Quick Build adds two toolbar buttons to IntelliJ IDEA for running the most common Maven lifecycle in a single click — no need to open the Maven tool window, type goals, or configure run configurations every time.

Toolbar Buttons

Button Mac Win / Linux Command
!m Skip Tests ⌘+⌥+S Ctrl+Alt+S mvn clean install -Dmaven.test.skip=true
m With Tests ⌘+⌥+M Ctrl+Alt+M mvn clean install

Features

  • Buttons appear in MainToolBar and NavBarToolBar — always one click away
  • Buttons are automatically disabled when no project is open
  • Buttons are hidden in non-Maven projects — no clutter
  • Keyboard shortcuts available out of the box — no configuration required
  • Uses the native MavenRunner API — output streams directly to the IDE's run console
  • The skip-tests variant never mutates global Maven settings — fully safe

Requirements

  • IntelliJ IDEA Community or Ultimate (2024.1 or later)
  • The open project must use Maven as its build system
